# Pakiet **SymPy**
Pakiet do obliczeń symbolicznych.<br/>
[Oficjalna strona internetowa](https://www.sympy.org/).<br/>
[Repozytorium GitHub](https://github.com/sympy/sympy).

## Instalacja
Windows: `py -m pip install -U sympy`<br/>
Linux / MacOS: `python3 -m pip install -U sympy`<br/>
[Więcej informacji](https://docs.sympy.org/latest/install.html).

## Materiały
Dokumentacja:
* [pełna dokumentacja](https://docs.sympy.org/latest/index.html),
* [przewodnik](https://docs.sympy.org/latest/tutorial/index.html),
* [Sympy: Symbolic Mathematics in Python](http://scipy-lectures.org/packages/sympy.html),
* [przegląd API](https://docs.sympy.org/latest/modules/index.html).

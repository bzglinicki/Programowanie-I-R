# Pakiet Matplotlib
Matplotlib to pakiet zawierający narzędzia wizualizacji danych dla języka Python.

Instalacja (więcej informacji [tutaj](https://matplotlib.org/stable/users/installing.html)):<br/>
`python -m pip install -U pip`<br/>
`python -m pip install -U matplotlib`

Oficjalna strona internetowa:<br/>
https://matplotlib.org/

Repozytorium GitHub:<br/>
https://github.com/matplotlib/matplotlib

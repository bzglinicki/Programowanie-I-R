# Pakiet **NumPy**
Pakiet wspierający obliczenia numeryczne. NumPy stanowi podstawę obliczeń naukowych w języku Python, jest bowiem wykorzystywany przez wiele innych pakietów dedykowanych temu zagadnieniu.<br/>
[Oficjalna strona internetowa](https://numpy.org/).<br/>
[Repozytorium GitHub](https://github.com/numpy/numpy).

## Instalacja
Windows: `py -m pip install -U numpy`<br/>
Linux / MacOS: `python3 -m pip install -U numpy`<br/>
[Więcej informacji](https://numpy.org/install/).

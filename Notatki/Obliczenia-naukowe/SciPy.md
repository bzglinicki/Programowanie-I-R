# Pakiet **SciPy**
Zaawansowany pakiet obliczeń naukowych, oparty na NumPy. Zawiera moduły związane m.in. z algebrą liniową, całkami, równaniami różniczkowymi zwyczajnymi, funkcjami specjalnymi, transformatami Fouriera, interpolacją.<br/>
[Oficjalna strona internetowa](https://scipy.org/scipylib/).<br/>
[Repozytorium GitHub](https://github.com/scipy/scipy).

## Instalacja
Windows: `py -m pip install -U scipy`<br/>
Linux / MacOS: `python3 -m pip install -U scipy`<br/>
[Więcej informacji](https://scipy.org/install.html).

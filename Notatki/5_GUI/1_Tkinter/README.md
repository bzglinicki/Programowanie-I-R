# Tkinter
Podstawowy pakiet do tworzenia GUI w języku Python, stanowi część biblioteki standardowej.<br/>
Oparty o [bibliotekę Tk](https://www.tcl.tk/), napisaną pierwotnie dla języka Tcl.<br/>
Wieloplatformowy.<br/>
Bezpłatny, również do użytku komercyjnego.<br/>
[Oficjalna strona internetowa](https://docs.python.org/3/library/tkinter.html).<br/>
[Repozytorium GitHub](https://github.com/python/cpython/tree/main/Lib/tkinter).

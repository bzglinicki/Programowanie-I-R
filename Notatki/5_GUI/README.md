# Graficzny interfejs użytkownika (GUI)
Napisanie aplikacji posiadającej graficzny interfejs użytkownika wymaga skorzystania z jednej z przeznaczonych do tego bibliotek.

[Lista i krótkie omówienie dostępnych pakietów.](https://wiki.python.org/moin/GuiProgramming)

## Wybrane pakiety do tworzenia graficznego interfejsu użytkownika.

### [Tkinter](./1_Tkinter)
Podstawowy pakiet do tworzenia GUI w języku Python, stanowi część biblioteki standardowej.
Oparty o [bibliotekę Tk](https://www.tcl.tk/), napisaną pierwotnie dla języka Tcl.

Wieloplatformowy. Bezpłatny, również do użytku komercyjnego.

### [PyQt](./2_PyQt)
Oparty o [bibliotekę Qt](https://www.qt.io/).

Wieloplatformowy. **Bezpłatny tylko do użytku niekomercyjnego**, zastosowanie komercyjne wymaga nabycia płatnej licencji.

### Inne

* [Kivy](https://kivy.org/),
* [Qt for Python (PySide)](https://www.qt.io/qt-for-python),
* [PyGObject (PyGTK)](https://pygobject.readthedocs.io/en/latest/),
* [wxPython](https://wxpython.org/),
* [pygame](https://www.pygame.org/) - informacje na temat tworzenia GUI dostępne są [tutaj](https://www.pygame.org/wiki/gui).

# PyQt
Oparty o [bibliotekę Qt](https://www.qt.io/).<br/>
Wieloplatformowy.<br/>
**Bezpłatny tylko do użytku niekomercyjnego**, zastosowanie komercyjne wymaga nabycia płatnej licencji.
[Oficjalna strona internetowa](https://riverbankcomputing.com/software/pyqt/).

## Instalacja
[Więcej informacji](https://riverbankcomputing.com/software/pyqt/download).

### PyQt6
Windows: `py -m pip install -U PyQt6`<br/>
Linux / MacOS: `python3 -m pip install -U PyQt6`

### PyQt5
Windows: `py -m pip install -U PyQt5`<br/>
Linux / MacOS: `python3 -m pip install -U PyQt5`
